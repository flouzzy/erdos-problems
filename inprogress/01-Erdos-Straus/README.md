# Erdős-Straus Conjecture

The Erdős-Straus conjecture states that for all integers $n \ge 2$, the rational number $4/n$ can be expressed as the sum of three positive unit fractions. That is, there exist positive integers $x, y$, and $z$ such that:

$$ \frac{4}{n} = \frac{1}{x} + \frac{1}{y} + \frac{1}{z} $$

This folder contains a comprehensive mathematical proof sketch and analysis of the conjecture, structured for autoformalization in Lean 4.
