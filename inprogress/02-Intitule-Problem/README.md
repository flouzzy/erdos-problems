# 02 - Erdős–Straus Conjecture

## Statement
The Erdős–Straus conjecture states that for every integer n ≥ 2, the rational number 4/n can be expressed as the sum of three unit fractions. That is, there exist positive integers x, y, and z such that 4/n = 1/x + 1/y + 1/z.

## Current Progress & Open Leads
While the conjecture has not been proven for all integers n, it has been verified computationally for all n up to a very large number (currently $10^{17}$).
- Many partial results exist, often showing that the conjecture holds for almost all n.
- Current approaches involve modular arithmetic, showing that counterexamples must have prime factors satisfying certain congruence conditions.
- Further research could involve extending the computational search or finding tighter modular constraints.
