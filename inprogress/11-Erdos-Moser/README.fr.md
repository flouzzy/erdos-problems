# 11 - Équation d'Erdős–Moser

## Énoncé
La conjecture de l'équation d'Erdős-Moser affirme que la seule solution entière à l'équation $1^k + 2^k + \dots + (m-1)^k = m^k$ pour $m > 1$ et $k > 1$ est la solution triviale $1^1 + 2^1 = 3^1$ (qui ne satisfait pas $k > 1$). En d'autres termes, il n'y a pas de solutions pour $k \ge 2, m \ge 2$.

Ce problème est actuellement **en cours**.
