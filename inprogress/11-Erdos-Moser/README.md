# 11 - Erdős–Moser Equation

## Statement
The Erdős–Moser equation conjecture states that the only integer solution to the equation $1^k + 2^k + \dots + (m-1)^k = m^k$ for $m > 1$ and $k > 1$ is the trivial solution $1^1 + 2^1 = 3^1$ (which does not satisfy $k > 1$). In other words, there are no solutions for $k \ge 2, m \ge 2$.

## Current Status
This problem is currently **in progress**.

We are exploring a deep analytical and algebraic approach using Faulhaber's formula and $p$-adic valuations to bound the potential solutions for large integers. A detailed proof sketch has been compiled.
