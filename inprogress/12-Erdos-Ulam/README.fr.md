# Problème d'Erdős-Ulam

Le problème d'Erdős-Ulam demande s'il existe un ensemble dense de points dans le plan euclidien tel que la distance entre deux points quelconques de l'ensemble soit un nombre rationnel.

Ce dossier contient une esquisse de preuve mathématique et une analyse de la conjecture, structurée pour une autoformalisation en Lean 4.

## Contenu
- `12-proof.pdf` : L'esquisse de preuve mathématique détaillée et l'analyse du problème d'Erdős-Ulam.
