# Erdős-Ulam Problem

The Erdős-Ulam problem asks whether there exists a dense set of points in the Euclidean plane such that the distance between any two points in the set is a rational number.

This folder contains a mathematical proof sketch and analysis of the conjecture, structured for autoformalization in Lean 4.

## Contents
- `12-proof.pdf`: The detailed mathematical proof sketch and analysis of the Erdős-Ulam problem.
